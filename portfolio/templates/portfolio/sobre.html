Sobre esta Aplicação

1. Arquitetura MVT do Django

A aplicação segue a arquitetura MVT (Model-View-Template) do Django:

  • Model — define a estrutura dos dados e comunica com a base de dados.
  • View — processa os pedidos do utilizador, interage com os models e devolve uma resposta.
  • Template — apresenta os dados em HTML que é enviado para o browser.
